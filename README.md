# F-Zero-Shot

## File Structure

- `agent.py`: constructor for agent
- `logger.py`: helper logger object
- `wrappers.py`: `gym` environment wrappers for data conversion
- `linear_model.py`: simple multi-layer perceptron model
- `main.py`: main driver script
